# newsletter-assets

Static image assets for the tastytrade internal hiring newsletter (loaded by email clients).
Do not rename or delete files — sent emails reference them by URL.

- `tt-logo.png` — tastytrade logo, 316×58 (renders 158×29)
- `raf-graphic.png` — hero illustration, 632×420 (renders 316×210)
- `chevron.png` — role-row chevron, 36×36 (renders 18×18)
